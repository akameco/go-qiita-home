# qiita-home

> Open Users Qiita Page


## Install

```
$ go get github.com/akameco/go-qiita-home
```

## CLI

Add an alias: `alias qh=go-qiita-home`

```
$ go-qiita-home --help

  Usage:
    qiita-home <username>
    qiita-home -h | --help
    qiita-home --version

```

## Usage

```
$ qh akameco
```
